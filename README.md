## ūüöÄ Features

- **Live Signal Processing**: Instant ASCII generation as you type.
- **Vast Font Library**: Access to numerous classic figlet fonts (Standard, Slant, Shadow, Block, etc.).
- **Smart Font Search**: quickly filter through fonts to find the perfect style for your terminal output.
- **Interactive UI**: A polished, terminal-inspired interface featuring scanlines, noise overlays, and CRT-style flickering effects.
- **Export Capabilities**:
  - **Copy to Clipboard**: Quick access for pasting into code comments or terminal headers.
  - **Text Download**: Export your generated art as a standard `.txt` file for documentation or READMEs.
- **Buffer History**: Keeps track of your recent generations for quick retrieval.
- **Adjustable Monitoring**: Fine-tune the output font size to fit your display requirements.
